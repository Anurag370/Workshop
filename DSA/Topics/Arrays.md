# Arrays

## Definition

An array is a contiguous block of memory storing elements of the same data type.

Declaration:
```cpp
// <datatype> <variable_name>[size] = {<values>}
int arr[6] = {1,2,3,4,5,6}; // Creates an array with elements.
int arr[10]; // Creates an array without any element but with size.
int arr[]; // Creates an array without specific size.
int arr[5] = {0}; //Creates an array filled with zero.

// Input from user
for(int i = 0; i < n; i++){
	cin >> arr[i];
}
```
